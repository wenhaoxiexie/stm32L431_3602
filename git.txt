项目提交命令：
        git add .
        git commit -m "text info"
        git push origin master


git 常用指令

1.查看所有分支

        git branch -a

1.1查看Log记录

        git log


2.新建分支

        git checkout -b 分支名

3.删除本地分支

        git branch -d 分支名

        或者

        git branch -D 分支名

4.删除远程分支

        git push origin --delete 分支名

5.切换分支

        git checkout 分支名

6.本地分支到远程

        git push origin 分支名:分支名

7.下拉远程分支

        git pull origin 远程分支名
